# AI Documentary Pipeline

Open-source AI workflow for cinematic documentary-style YouTube production.

## Overview

AI Documentary Pipeline is an open-source toolkit designed to automate and streamline long-form knowledge and documentary-style video production.

This project combines:
- OpenAI Image generation workflows
- Remotion-based video rendering
- AI narration pipelines
- Prompt automation systems
- Batch rendering workflows
- YouTube documentary production tools

## Features

- Cinematic AI image prompt generation
- Documentary-style visual pipeline
- Remotion animation workflow
- Automated scene generation
- Voiceover integration
- Subtitle generation
- Batch rendering support
- Long-form YouTube optimization

## Workflow

1. Generate documentary script
2. Create cinematic prompts
3. Generate AI visuals
4. Render scenes with Remotion
5. Add narration and subtitles
6. Export final YouTube-ready video
