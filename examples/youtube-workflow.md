# YouTube Documentary Workflow

This starter workflow outlines an open-source AI documentary production pipeline using prompts, narration, subtitles, Remotion scenes, and YouTube export steps.

1. Research the topic and collect source notes.
2. Generate a clear documentary script.
3. Break the script into visual beats.
4. Write cinematic AI image prompts for each beat.
5. Generate and review image assets.
6. Build animated scenes in Remotion.
7. Add narration, subtitles, and background music.
8. Render the final video.
9. Export a YouTube-ready file with title, description, and thumbnail notes.

Example output structure:

```text
assets/
  images/
  narration/
prompts/
  spacex-launch-prompt.md
remotion/
  basic-scene.tsx
exports/
  youtube-final.mp4
```
